Detailed Description

Cette classe s'occupe de l'affiche de la voiture, des obstacle et du score.
"texture_V" et "texture_V2" sont des données membres de type pointeur de SDL_Texture qui sont utilisé pour afficher les deux images de la voiture.
"texture_Obs" est un donnée membre de type pointeur de SDL_Texture qui est utilisé pour afficher l'image pour les obstacles.
"texture_R0", "texture_R1", "texture_R2", "texture_R3" ont des données membres de type pointeur de SDL_Texture qui sont utilisé pour afficher les quatres images de la route.
"Indice_Texture_R": un int qui se balade entre 0 inclus et 3 inclus et qui décide quelle image de la route on affiche sur l'écran pour avoir un effet de route qui défile.
