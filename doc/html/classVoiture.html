Detailed Description

La classe voiture contient un donnée membre de type pointeur de classe Position ("Voiture_Position") et un donnée membre "Deplacement" qui est une variable de type int et qui décide de combien de pixel se déplace la voiture pour aller à gauche/droite/haut/bas.
